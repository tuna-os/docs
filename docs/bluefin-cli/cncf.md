*A comprehensive collection of Cloud Native tools.*

**Core & Management**
- [argo](https://argoproj.io): Get stuff done with container-native workflows for Kubernetes
- [cilium-cli](https://cilium.io): CLI to install, manage & troubleshoot Kubernetes clusters running Cilium
- [coredns](https://coredns.io/): DNS server that chains plugins
- [crossplane](https://github.com/crossplane/crossplane): Build control planes without needing to write code
- [dapr-cli](https://dapr.io): Client for Dapr
- [cri-tools](https://github.com/kubernetes-sigs/cri-tools): CLI and validation tools for Kubelet Container Runtime Interface (CRI)
- [envoy](https://www.envoyproxy.io/index.html): Cloud-native high-performance edge/middle/service proxy
- [falco](https://github.com/ysugimoto/falco): VCL parser and linter optimized for Fastly
- [flux](https://www.influxdata.com/products/flux/): Lightweight scripting language for querying databases
- [harbor-cli](https://github.com/goharbor/harbor-cli): CLI for Harbor container registry
- [helm](https://helm.sh/): Kubernetes package manager
- [istioctl](https://istio.io/): Istio configuration command-line utility
- [kn](https://github.com/knative/client): Command-line interface for managing Knative Serving and Eventing resources
- [kubernetes-cli](https://kubernetes.io/docs/reference/kubectl/): Kubernetes command-line interface
- [linkerd](https://linkerd.io): Command-line utility to interact with linkerd
- [opa](https://www.openpolicyagent.org): Open source, general-purpose policy engine
- [prometheus](https://prometheus.io/): Service monitoring system and time series database
- [tiup](https://www.pingcap.com): TiDB is a MySQL compatible distributed database, and tiup is a component manager for testing and using TiDB locally
- [vitess](https://vitess.io): Database clustering system for horizontal scaling of MySQL
- [cmctl](https://cert-manager.io): Command-line tool to manage cert-manager
- [nerdctl](https://github.com/containerd/nerdctl): ContaiNERD CTL - Docker-compatible CLI for containerd
- [etcd](https://github.com/etcd-io/etcd): Key value store for shared configuration and service discovery

**Additional Tools**
- [ah](https://github.com/artifacthub/hub): Artifact Hub command line tool
- [pack](https://github.com/buildpacks/pack): A CLI for building apps using Cloud Native Buildpacks
- [c7n](https://github.com/cloud-custodian/cloud-custodian): Rules engine for cloud security, cost optimization, and governance
- [cortex](https://cortexmetrics.io/): Long term storage for Prometheus
- [karmadactl](https://karmada.io/): CLI for Karmada control plane
- [kubevela](https://kubevela.io): Application Platform based on Kubernetes and Open Application Model
- [virtctl](https://kubevirt.io/): Allows for using more advanced kubevirt features
- [kubescape](https://kubescape.io): Kubernetes testing according to Hardening Guidance by NSA and CISA
- [kyverno](https://kyverno.io/): Kubernetes Native Policy Management
- [lima](https://lima-vm.io/): Linux virtual machines
- [litmusctl](https://litmuschaos.io): Command-line interface for interacting with LitmusChaos
- [nats-server](https://nats.io): Lightweight cloud messaging system
- [notation](https://notaryproject.dev/): CLI tool to sign and verify OCI artifacts and container images
- [openfga](https://openfga.dev/): High performance and flexible authorization/permission engine
- [opentelemetry-cpp](https://opentelemetry.io/): OpenTelemetry C++ Client
- [operator-sdk](https://sdk.operatorframework.io/): SDK for building Kubernetes applications
- [thanos](https://thanos.io): Highly available Prometheus setup with long term storage capabilities
- [grpc](https://grpc.io/): Next generation open source RPC library and framework
