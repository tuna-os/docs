# TunaOS Flatpaks

Install GNOME office apps directly from our Flatpak remote.

## Add the remote

```bash
flatpak remote-add tuna-os https://docs.tuna-os.org/flatpak/tuna-os.flatpakrepo
```

## Available apps

| App | Install |
|-----|---------|
| [🧮 Tables](/docs/apps/tables) | `flatpak install tuna-os org.tunaos.tables` |
| [🎬 Decks](/docs/apps/decks) | `flatpak install tuna-os org.tunaos.decks` |
| [📄 Letters](/docs/apps/letters) | `flatpak install tuna-os org.tunaos.letters` |
