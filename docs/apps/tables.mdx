# 🧮 Tables

Spreadsheet application with ~400 Excel-compatible functions via [HyperFormula](https://hyperformula.handsontable.com/).  Powered by [Jspreadsheet CE](https://jspreadsheet.com/) for the grid.

Reads and writes CSV, XLSX, and ODS files.

## Install

<CodeBlock language="bash">
flatpak remote-add tuna-os https://docs.tuna-os.org/flatpak/tuna-os.flatpakrepo
flatpak install tuna-os org.tunaos.tables
</CodeBlock>

## Features

- **Formulas**: SUM, IF, VLOOKUP, AVERAGE, and ~400 more
- **Multi-sheet**: workbook tabs with Ctrl+PgUp/PgDn navigation
- **Formatting**: Bold, Italic, Underline, Alignment, Borders
- **Charts**: Bar, Line, Pie via openpyxl
- **Sort & Filter**: Column headers with sort arrows and filter dropdowns
- **Data tools**: Merge cells, freeze panes, number formatting, fill down/right
- **Keyboard shortcuts**: Excel-compatible (Ctrl+B/I/U, Ctrl+D/R, Ctrl+Space, etc.)
