# Tables

Spreadsheet with ~400 Excel-compatible functions

Part of the [TunaOS](https://github.com/tuna-os) GNOME office suite.

## Install

```bash
flatpak remote-add tuna-os https://docs.tuna-os.org/flatpak/tuna-os.flatpakrepo
flatpak install tuna-os org.tunaos.tables
```

## Features

See [README](https://github.com/hanthor/tables#readme) for full details.

## Build from source

```bash
git clone https://github.com/hanthor/tables.git
cd tables
just setup
just build
just run
```
