# 🍺 Tavern

AI-powered developer companion for the GNOME desktop.  Chat with LLMs, manage agents, and automate workflows — all in a native GTK4/libadwaita app.

## Install

<CodeBlock language="bash">
flatpak remote-add tuna-os https://docs.tuna-os.org/flatpak/tuna-os.flatpakrepo
flatpak install tuna-os dev.hanthor.Tavern
</CodeBlock>
