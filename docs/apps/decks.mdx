# 🎬 Decks

Presentation application powered by [Fabric.js](http://fabricjs.com/) for canvas editing and [Reveal.js](https://revealjs.com/) for fullscreen presentations.

Reads and writes PPTX and ODP files.  Exports to PDF.

## Install

<CodeBlock language="bash">
flatpak remote-add tuna-os https://docs.tuna-os.org/flatpak/tuna-os.flatpakrepo
flatpak install tuna-os org.tunaos.decks
</CodeBlock>

## Features

- **Canvas editing**: Text boxes, shapes, images on a Fabric.js canvas
- **Slide management**: Add, delete, duplicate, reorder via sidebar
- **Layouts**: Blank, Title, Title+Content, Two-Column presets
- **Transitions**: Fade, Slide, Convex, Concave, Zoom (per-slide)
- **Present mode**: Fullscreen with B/W/. blank shortcuts
- **Undo/Redo**: 30-level canvas action history
- **Export**: Multi-page PDF
