# Decks

Presentation app with slide editing and fullscreen mode

Part of the [TunaOS](https://github.com/tuna-os) GNOME office suite.

## Install

```bash
flatpak remote-add tuna-os https://docs.tuna-os.org/flatpak/tuna-os.flatpakrepo
flatpak install tuna-os org.tunaos.decks
```

## Features

See [README](https://github.com/hanthor/decks#readme) for full details.

## Build from source

```bash
git clone https://github.com/hanthor/decks.git
cd decks
just setup
just build
just run
```
