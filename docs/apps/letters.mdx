# Letters

Modern word processor for the GNOME desktop

Part of the [TunaOS](https://github.com/tuna-os) GNOME office suite.

## Install

```bash
flatpak remote-add tuna-os https://docs.tuna-os.org/flatpak/tuna-os.flatpakrepo
flatpak install tuna-os org.tunaos.letters
```

## Features

See [README](https://github.com/hanthor/letters#readme) for full details.

## Build from source

```bash
git clone https://github.com/hanthor/letters.git
cd letters
just setup
just build
just run
```
