# 📄 Letters

Modern, minimalist word processor for the GNOME desktop.

> **Hard fork** of [Letters by Satvik Patwardhan](https://codeberg.org/eyekay/letters).  All credit for the original architecture and design goes to Satvik.

Reads and writes DOCX, ODT, MD, HTML, TXT, and RTF via [pandoc](https://pandoc.org).  Exports to PDF via [WeasyPrint](https://weasyprint.org/).

## Install

<Tabs>
  <TabItem value="python" label="Python (stable)">
    <CodeBlock language="bash">

## Features

- **Rich formatting**: Bold, Italic, Underline, Strikethrough, Highlight
- **Styles**: Paragraph, H1–H6, Code, Blockquote
- **Alignment**: Left, Center, Right, Justify (Ctrl+L/E/R/J)
- **Find & Replace**: Ctrl+F / Ctrl+H
- **Tables**: Insert HTML tables
- **Word count**: Live status bar
- **Multiple tabs**: Ctrl+T, Ctrl+W
- **Keyboard shortcuts**: Docs/Word-compatible
